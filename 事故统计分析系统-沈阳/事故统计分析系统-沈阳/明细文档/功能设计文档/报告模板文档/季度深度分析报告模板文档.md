# 季度深度分析报告模板文档

## 说明

本模板基于《2025年全市生产安全事故统计分析报告》真实公文结构扩展整理，适用于系统内 `季度深度分析报告` 模板自动生成。

模板特点：

- 延续正式报告写法，适合局领导审阅和季度分析汇报
- 比月报多出“季度内月份分布”“重点行业专题分析”“阶段性工作建议”
- 支持同比、环比、异常点说明、重点行业和重点区域深度解读

---

## 三、系统填充字段表

| 字段分类 | 字段名称 | 说明 |
|----------|----------|------|
| 基础信息 | `{YYYY}` `{Q}` | 报告年份、季度 |
| 总量指标 | `{事故起数}` `{死亡人数}` `{重伤人数}` | 季度汇总值 |
| 对比指标 | `{同比事故变化描述}` `{环比事故变化描述}` | 如“同比减少5起、下降12.5%” |
| 趋势指标 | `{月均事故}` `{高发月份}` `{低发月份}` | 季度内月份走势 |
| 行业指标 | `{道路运输起数}` `{建筑业起数}` `{商贸制造起数}` | 重点行业汇总 |
| 类型指标 | `{高发类型1}` `{高发类型2}` | 季度高发类型 |
| 区域指标 | `{重点地区1}` `{重点地区2}` `{重点地区3}` | 重点区域 |
| 结论字段 | `{特点1}` `{特点2}` `{特点3}` `{特点4}` | 由规则或 AI 生成 |
