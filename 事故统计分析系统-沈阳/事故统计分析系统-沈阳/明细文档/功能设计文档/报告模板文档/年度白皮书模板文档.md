# 年度白皮书模板文档

## 说明

本模板基于《2025年全市生产安全事故统计分析报告》真实年度样例整理，适用于系统内 `年度白皮书` 模板自动生成。

模板特点：

- 完整保留真实年度报告的正式结构
- 适合安委办、应急管理局、局领导年度汇报和归档发布
- 覆盖“总体情况、重点行业领域情况、主要特点及工作建议”三大部分

#### （二）预警研判和工作建议

各地区、各有关部门要充分清醒认识安全生产面临的复杂形势，按照市委、市政府工作部署，强化安全监管责任落实，督促生产经营单位全面落实安全生产主体责任，有效防范和减少各类事故发生。重点抓好以下工作：

1. 进一步提高政治站位，统筹抓好重点行业领域安全生产工作。
2. 加强道路运输安全综合整治，强化人、车、路协同治理。
3. 加强建筑施工领域安全管理，紧盯危大工程、装饰装修、电气焊等重点环节。
4. 持续推进城镇燃气、供暖、危险化学品、烟花爆竹、矿山、工贸、特种设备、文旅、农业农村、消防等重点领域安全风险管控。
5. 聚焦节假日、极端天气、复工复产和重点活动保障等关键时段，提前做好风险预警研判和应急准备。

## 三、系统填充字段表

| 字段分类 | 字段名称 | 说明 |
|----------|----------|------|
| 基础信息 | `{YYYY}` | 报告年份 |
| 总量指标 | `{事故起数}` `{死亡人数}` `{重伤人数}` | 年度汇总指标 |
| 对比指标 | `{同比事故变化描述}` `{同比死亡变化描述}` | 与上年对比结果 |
| 季度/月度指标 | `{季度平均事故}` `{月均事故}` | 年度趋势分析口径 |
| 行业指标 | `{道路运输起数}` `{商贸制造起数}` `{建筑业起数}` | 重点行业指标 |
| 类型指标 | `{高发类型1}` `{高发类型2}` `{高发类型3}` | 年度高发事故类型 |
| 地区指标 | `{重点地区1}` `{重点地区2}` `{重点地区3}` | 年度事故高发区域 |
| 特点与建议 | `{特点1}` `{特点2}` `{年度总体判断}` | 由规则或 AI 生成 |
