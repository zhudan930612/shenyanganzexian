# 月度简报模板文档

## 说明

本模板基于《2025年全市生产安全事故统计分析报告》真实公文结构整理，适用于系统内 `月度简报` 模板自动生成。

模板特点：

- 延续安委办正式报告的行文风格
- 采用“总体情况 → 重点行业 → 风险提示和建议”的结构
- 便于系统自动填充事故起数、死亡人数、行业分布、区域分布、同比/环比等字段

---

## 一、标题与导语模板

### 标题

`{YYYY}年{M}月全市生产安全事故统计分析简报`

### 导语

为全面掌握 `{YYYY}年{M}月` 全市生产安全事故发生的规律特点，有针对性地采取防范措施，有效防控各类生产安全亡人事故，现对本月全市事故统计数据进行分析，具体如下：

## 三、系统填充字段表

| 字段分类 | 字段名称 | 说明 |
|----------|----------|------|
| 基础信息 | `{YYYY}` `{M}` | 报告年份、月份 |
| 总量指标 | `{事故起数}` `{死亡人数}` `{重伤人数}` | 全市月度汇总 |
| 对比指标 | `{同比事故变化描述}` `{环比事故变化描述}` | 如“同比减少2起、下降8.3%” |
| 重点事故 | `{较大事故起数}` `{重特大事故起数}` | 月内重大事故情况 |
| 行业分布 | `{道路运输起数}` `{建筑业起数}` `{商贸制造起数}` | 重点行业统计值 |
| 类型分布 | `{高处坠落起数}` `{火灾起数}` `{触电起数}` | 事故类型统计值 |
| 地区分布 | `{重点地区1}` `{重点地区2}` `{重点地区3}` | 月内高发区域 |
| 结论字段 | `{总体趋势判断}` `{高发事故类型结论}` `{特点1}` | 由规则或 AI 生成 |
